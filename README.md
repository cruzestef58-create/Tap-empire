# ⛰️ Tap Empire

Un petit jeu de type *Clicker / Idle* : fais évoluer ta civilisation de l'Âge de Pierre
à l'Ère Industrielle, un clic à la fois.

## 🎮 Jouer

➡️ **[Cliquez ici pour jouer](https://TON-PSEUDO.github.io/tap-empire/)**

*(Remplace `TON-PSEUDO` par ton pseudo GitHub une fois Pages activé.)*

## ✨ Fonctionnalités

- Clic sur le minerai pour produire des points (particules + son).
- Bâtiments à production passive (PPS).
- Amélioration de la puissance de clic.
- **5 ères** à débloquer, chacune avec sa propre ambiance, ses bâtiments et un
  bonus de production global : Âge de Pierre → Bronze → Fer → Antiquité → Ère Industrielle.
- Sauvegarde automatique dans le navigateur.

## 🛠️ Détails techniques

- 100 % HTML / CSS / JavaScript, sans dépendance.
- Fichier unique et autonome : toutes les icônes vectorielles (SVG) sont intégrées dans `index.html`.
- Pour jouer en local : il suffit d'ouvrir `index.html` (aucun autre fichier requis).
